Pipeline

1. Runtime Capture

eBPF hooks collect TLS handshake metadata (SNI, groups, negotiated group, success) with minimal overhead.

2. CryptoBOM

JSONL → CryptoBOM v2.1 observations hashed with bom_ref for deduplication and temporal diffing.

3. Policy Gate

OPA evaluates PQC migration & crypto hygiene (RSA key size, legacy hashes, hybrid readiness, tagging).

4. Attestation & API

FastAPI service signs (roadmap) aggregated risk posture and emits human + machine readable artifacts.

Quick Start

# build eBPF + loader
cd carnot-agent/ebpf-core
make && cd go-loader && go build -o bin/carnot-ebpf-loader ./...
./bin/carnot-ebpf-loader -obj ../openssl_handshake.bpf.o -out runtime.jsonl -libssl /lib/x86_64-linux-gnu/libssl.so.3

# convert to CryptoBOM
python integrations/runtime/ebpf_to_bom.py --in runtime.jsonl --out runtime.bom.json --asset-id host:demo

# run policy gate (opa)
opa eval --format pretty -i runtime.bom.json -d policies 'data'

# attestation (future combined input)
uvicorn api.main:app --reload
